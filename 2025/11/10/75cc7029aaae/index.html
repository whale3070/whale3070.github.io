著名的solidity合约

以太坊中存在哪些著名的solidity合约？

以太坊生态中已经出现了大量“经典级”“教材级”的 Solidity 合约，它们不仅重要，而且是学习智能合约与审计的最佳素材。下面我从基础协议、DeFi、DAO、基础设施、NFT、预言机、账户体系等维度为你系统列出最值得学习的 Solidity 合约。

这些合约都是在以太坊历史上具有深远影响的“标杆项目”。

1. Ethereum 官方与基础协议（必须掌握）

这些是智能合约世界的“标准库”和基础设施。

1) ERC-20 标准

最基础的代币标准。
代表合约：

  • OpenZeppelin ERC20.sol
  • USDT（早期部分）
  • USDC（中心化版本）

2) ERC-721（NFT 标准）

NFT 的基础协议。
代表合约：

  • OpenZeppelin ERC721.sol
  • CryptoKitties（NFT 爸爸级项目）

3) ERC-1155（Multi-token 标准）

游戏资产常用。
代表合约：

  • OpenZeppelin ERC1155.sol
  • Enjin

4) ENS（Ethereum Name Service）

以太坊的 DNS，核心合约极具研究价值。

2. DeFi 领域（审计必看）

1) Uniswap v2 / v3

自动做市 AMM 协议，DeFi 的“数学圣经”级源码。

  • UniswapV2Pair.sol（AMM 核心）
  • UniswapV3Pool.sol（集中流动性）
  • UniswapV3Factory.sol

2) Aave v2/v3

最经典的借贷协议之一。

代表合约：

  • LendingPool.sol
  • StableDebtToken.sol
  • VariableDebtToken.sol
  • AToken.sol

3) Compound v2

借贷协议的基础设计框架。

代表合约：

  • Comptroller.sol
  • CErc20.sol
  • InterestRateModel.sol

4) MakerDAO

最强的去中心化稳定币系统。

代表合约：

  • Vat.sol
  • Vow.sol
  • Cat.sol
  • Dog.sol
  • Jug.sol
    （设计极其复杂但值得深入研究）

5) Curve Finance

稳定币 AMM。
特点：自定义数学曲线模型。

3. 去中心化治理（DAO 合约）

1) Aragon DAO 框架

复杂度极高，治理逻辑可扩展。

2) Compound Governor Alpha / Bravo

现在 DAO 世界最常用的投票治理框架。

3) ENS DAO 治理模型

代表性强，安全性高。

4. NFT 生态经典合约

1) CryptoKitties

世界上第一个爆火的 NFT 项目（可繁殖 NFT）。

2) BAYC（Bored Ape Yacht Club）

ERC721A、YGOR 标准，主线简单但安全要点多。

3) Azuki（ERC721A）

最著名的“批量铸造优化”合约。

4) Loot

完全链上随机生成 NFT 的鼻祖。

5. 预言机（Oracle）合约

1) Chainlink AggregatorV3Interface

以太坊上最常用的价格预言机。

核心文件：

  • AggregatorV3.sol
  • VRF（链下随机数）

通过这些学习你可以掌握：

  • 确权机制
  • 去中心化喂价网络

6. Layer2 / Rollup

1) Optimism Bedrock

L2 Sequencer + Fraud Proof logic

2) Arbitrum Nitro

复杂但值得理解 Fraud Proof 的结构

3) zkSync

使用 Solidity + Circuits + zkEVM 的组合结构。

7. 基础设施（红蓝队审计必看）

1) Gnosis Safe（多签钱包）

全球最常用的资产管理合约，审计必修课。

2) OpenZeppelin Contracts

Solidity 的标准库，任何合约开发者必须掌握。

包含模块：

  • Ownable
  • AccessControl
  • ReentrancyGuard
  • ERC20 / 721 / 1155
  • Pausable
  • UUPSUpgradeable

3) Multicall

允许在一次交易中调用多个合约。

8. DEX / MEV / Staking

1) SushiSwap（Uniswap fork）

包含 SUSHI 奖励模型。

2) Balancer

多资产加权池 AMM。

3) Lido Staking / wstETH

质押 ETH 的代表项目。

关键点：

  • Rebase token 机制
  • 权益再质押

9. 模块型示例（方便新手阅读）

如果你是 Solidity 学习初期，我建议你从以下阅读难度最低的入手：

✅ Level 0（入门）

  • OpenZeppelin ERC20.sol
  • OpenZeppelin Ownable.sol
  • OpenZeppelin ReentrancyGuard.sol

✅ Level 1（进阶）

  • UniswapV2Pair.sol（AMM）
  • GnosisSafe.sol（多签钱包）
  • Chainlink Price Feed（预言机）

✅ Level 2（高级）

  • UniswapV3Pool.sol（数学密度大）
  • MakerDAO Vat.sol（国家级金融系统）
  • Aave LendingPool.sol（复杂状态机）

小结：以太坊最值得阅读的合约 Top 10（我推荐顺序）

如果你只挑 10 个，我建议：

  1. OpenZeppelin ERC20
  2. OpenZeppelin Ownable
  3. OpenZeppelin AccessControl
  4. Uniswap V2 Pair
  5. Uniswap V3 Pool
  6. Gnosis Safe
  7. Chainlink PriceFeed
  8. Aave LendingPool
  9. MakerDAO Vat
  10. ERC721 + ERC721A（Azuki）

这是 Solidity 学习和审计的黄金组合。

✅ 哪里还能看部署在主网的真实智能合约？

除了 GitHub，你也可以在 Etherscan 上看到真实部署的合约源码。

步骤：

  1. 打开 https://etherscan.io

  2. 搜索任意项目
    例如：USDC、Uniswap、Aave

  3. 点 Contract → 你能看到：

    • 已验证源码
    • ABI
    • 状态变量
    • 事件
    • 历史交易
    • 合约读/写界面

这是做审计必须掌握的技能。
