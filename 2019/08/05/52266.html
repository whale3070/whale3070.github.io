DockerпјҲдёҖпјүд»Җд№ҲжҳҜdocker

еҸӮиҖғпјҡDocker е…Ҙй—Ёж•ҷзЁӢ

? д»Җд№ҲжҳҜdocker

е®ғжҳҜдёҖз§ҚиҷҡжӢҹеҢ–жҠҖжңҜпјҢжҳҜдёәдәҶи§ЈеҶіиҪҜд»¶е…је®№й—®йўҳиҖҢз”ҹзҡ„гҖӮ

дјҡLinuxзҡ„еҗҢеӯҰйғҪзҹҘйҒ“пјҢжңүж—¶еҖҷдҪ е®үиЈ…дёҖдёӘиҪҜд»¶пјҢе®ғйңҖиҰҒжҹҗдёӘеә“пјҢдәҺжҳҜдҪ е°ұдјҡиҫ“е…ҘвҖңapt-get install xxxвҖқпјҢжңүж—¶еҖҷе®ғйңҖиҰҒдёҚжӯўдёҖдёӘдҫқиө–пјҢиҖҢдё”жңүж—¶еҖҷдјҡжҠҘеҗ„з§Қеҗ„ж ·зҡ„й”ҷиҜҜгҖӮжңүдәӣиҪҜд»¶йңҖиҰҒpythonзҺҜеўғгҖҒжңүдәӣиҪҜд»¶йңҖиҰҒjavaзҺҜеўғгҖҒжңүдәӣйңҖиҰҒ.netжЎҶжһ¶гҖӮ

еҰӮжһңжңүдёҖдёӘзі»з»ҹпјҢеә”з”ЁиҪҜд»¶е·Із»Ҹжҗӯе»әеҘҪдәҶгҖӮеңЁдҝЎжҒҜжҠҖжңҜдёӯпјҢdockerе°ұжҳҜиҝҷж ·дёҖз§ҚжҠҖжңҜгҖӮз”ЁжҲ·еҸҜд»ҘвҖқејҖзӣ–еҚійЈҹвҖңпјҢзұ»дјјдәҺзңҹз©әеҢ…иЈ…зҶҹйЈҹзҡ„иҝҷж ·дёҖз§ҚжҠҖжңҜгҖӮ

пјҹ dockerдёҺVirtual Machineзҡ„еҢәеҲ«

dockerжҠҖжңҜејҖе§ӢдәҺ2013е№ҙпјҢиҖҢиҷҡжӢҹеҢ–жҠҖжңҜеңЁ20дё–зәӘ70е№ҙд»Је·Із»ҸеңЁеӨ§еһӢжңәдёҠеә”з”ЁгҖӮ

dockerзҡ„жңҖе°ҸеҚ•дҪҚжҳҜеә”з”ЁвҖ”вҖ”applicationпјҢVMзҡ„жңҖе°ҸеҚ•дҪҚжҳҜж“ҚдҪңзі»з»ҹвҖ”вҖ”operation system.

е®ғжҳҜlinuxе®№еҷЁпјҢдёҖз§ҚиҷҡжӢҹеҢ–жҠҖжңҜпјҢе°Ҷи®Ўз®—жңәзҡ„зЎ¬д»¶иө„жәҗжҠҪиұЎеҮәжқҘпјҢжЁЎжӢҹдёәеӨҡеҸ°и®Ўз®—жңәгҖӮ
е®ғдёҺиҷҡжӢҹжңәдёҚеҗҢпјҢиҷҡжӢҹжңәжҳҜе®Ңе…ЁжЁЎжӢҹпјҲжңүе®Ңж•ҙзҡ„ж“ҚдҪңзі»з»ҹзӯүпјүпјҢиҖҢDockerжҳҜзӣҙжҺҘи°ғз”ЁзЎ¬д»¶еҜ№ж“ҚдҪңзі»з»ҹзҡ„жҺҘеҸЈпјҢж— йңҖдёӯй—ҙеұӮпјҢйҖҹеәҰиҫҫеҲ°дәҶз§’зә§гҖӮ

пјҹд»Җд№ҲжҳҜдә‘и®Ўз®—

дә‘и®Ўз®—пјҲCloud ComputingпјүжҳҜеҲҶеёғејҸи®Ўз®—пјҲDistributed ComputingпјүгҖҒе№¶иЎҢи®Ўз®—пјҲParallel ComputingпјүгҖҒж•Ҳз”Ёи®Ўз®—пјҲUtility ComputingпјүгҖҒ зҪ‘з»ңеӯҳеӮЁпјҲNetwork Storage TechnologiesпјүгҖҒиҷҡжӢҹеҢ–пјҲVirtualizationпјүгҖҒиҙҹиҪҪеқҮиЎЎпјҲLoad BalanceпјүгҖҒзғӯеӨҮеҶ—дҪҷпјҲHigh Availableпјүзӯүдј з»ҹи®Ўз®—жңәе’ҢзҪ‘з»ңжҠҖжңҜеҸ‘еұ•иһҚеҗҲзҡ„дә§зү©гҖӮ

жңӘжқҘзҡ„дә‘и®Ўз®—пјҢе°ұеғҸжҲ‘д»¬дҪҝз”Ёж°ҙз”өз…Өж°”дёҖж ·пјҢжҲ‘д»¬д»ҺжқҘдёҚдјҡжғізқҖеҺ»е»әз”өеҺӮпјҢд№ҹдёҚе…іеҝғз”өеҺӮеңЁе“ӘйҮҢпјҢеҸӘиҰҒжҸ’дёҠжҸ’еӨҙпјҢе°ұиғҪз”Ёз”өгҖӮ

зңҹжӯЈзҡ„дә‘и®Ўз®—дёҖе®ҡжңүдёӨдёӘйҮҚиҰҒзҡ„еҹәзЎҖжқЎд»¶пјҡ

  1. и®Ўз®—иө„жәҗзҡ„иҷҡжӢҹеҢ–пјҢжҢүи®Ўз®—иғҪеҠӣиҙӯд№°жүҚжҳҜзңҹжӯЈзҡ„дә‘и®Ўз®—пјҢзӣ®еүҚзҡ„иҷҡжӢҹжңәеҳӣпјҢе—ҜпјҢзңӢдёҚеҲ°жңәеҷЁеҝғйҮҢдёҚиёҸе®һгҖӮ
  2. дә‘и®Ўз®—иғҪеҠӣзҡ„еј№жҖ§дјёзј©иғҪеҠӣпјҢдёҖе®ҡжҳҜжғіз”ЁеӨҡе°‘е°ұжҳҜеӨҡе°‘пјҢдёҚз”Ёзҡ„ж—¶еҖҷе°ұдёҚиҰҒпјҢзӣ®еүҚзҡ„дё“жңүдә‘еҳӣпјҢе—ҜпјҢжңәеҷЁдёҚж”ҫеңЁжҲ‘иҝҷйҮҢеҝғйҮҢд№ҹдёҚиёҸе®һгҖӮ(жң¬ж®өиҪ¬иҪҪиҮӘзҹҘд№Һ)
