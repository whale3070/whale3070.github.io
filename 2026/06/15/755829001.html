时代红利是电梯，不是你的腿：为什么长辈的"成功经验"正在害死年轻人

怎么看朱虹的三个学位

1989年本科：水文地质与工程地质（湖南科技大学）

这是一个完全不相关的理工科起点。80年代末大学生分配工作是常态，留校任教在当时不算难，不代表学术能力突出。

1993-1996年硕士：社会学（南京大学）

本科学地质的人转社会学，这个跨度很大。90年代跨专业考研竞争远不如现在激烈，社会学在当时也不是热门学科。能考上南大当然说明有一定能力，但这个转型也意味着她的社会学基础是从硕士才开始的，不是科班出身。

1999-2002年博士：社会心理学（南京大学）

硕士毕业后隔了三年才读博，可能是在职攻读。2000年前后的博士录取率和毕业标准与今天不可同日而语。那个年代博士名额多、竞争相对温和，CSSCI论文发表的难度也远低于现在。

整体判断：

她的学术路径是一个典型的"时代红利型"学者——赶上了中国高校扩张期，在竞争不充分的环境下完成了学历积累。这不是说她完全没能力，但她的头衔含金量需要放在历史背景下去理解，不能和今天的同等学位等量齐观。

看她的论文列表也能印证：核心研究方向是消费心理、品牌营销、社会信任，发表量尚可但缺乏国际期刊的高影响力成果（SSCI论文极少，主要是国内CSSCI）。这在南大商学院这个平台上，属于中规中矩，谈不上顶尖。

开篇：一个油画生的三次换乘

我本科学油画。

2018年，我休学去北京做网络安全实习。2019年拿到深信服安全研究岗的offer。2020年毕业，在网络安全行业干了三年。2023年离开360后，我学雅思到6.5，拿了香港身份。2026年，我在Web3交易所做安全工程师。

如果用传统眼光看，这条路简直是胡来——学油画的搞安全？不考研不考公？放弃大陆社保？每一步都是”不靠谱”。

但如果用”时代红利”的框架看，每一步都踩在了窗口期上：

  • 2018-2019年：网络安全行业爆发期，人才缺口巨大，门槛低到非科班出身也能进
  • 2020-2022年：安全行业薪资红利期，积累了技能和行业资历
  • 2023-至今：Web3需要安全人才，而传统安全从业者很少有人跨过来

我不是天才，不是学霸，本科画了四年油画。我只是在对的时间站到了对的位置。

这就是时代红利——你不需要是最优秀的人，只需要在窗口打开时走进去。

第一部分：什么是时代红利

时代红利的定义很简单：在特定历史窗口期，某个领域的机会对参与者异常慷慨，门槛低、回报高、先到先得。

它有几个核心特征：

1. 窗口期有限

我2018年进网络安全行业时，行业刚起步，公司抢人。到了2023年，同样的岗位已经卷到要求985硕士+CTF获奖经历。窗口开了三四年，然后关上了。

2. 门槛低得不像话

我一个油画生，靠自学能进安全行业。放在今天，同样的背景连简历筛选都过不了。红利期的特征就是：行业增速远快于人才供给，所以不得不降低门槛。

3. 事后被包装成”个人奋斗”

如果我现在说”我靠自己的努力从油画转行到安全再转Web3”，听起来很励志对吧？但真相是：如果我晚两年才开始，同样的努力大概率一事无成。

时代红利 = 电梯。你站上去了，它把你带到高处。但电梯不是你的腿，你不能把”被电梯带上去”等同于”自己爬上去”。

第二部分：过去四十年，中国有哪些时代红利

时期 红利 受益者 门槛
1980s-90s 个体经济起步 最早下海经商的人 敢干就行
1990s-2000s 高校扩张+留校容易 那一代学者 考上就行
1998-2015 房价单边上涨 所有早期买房者 凑得出首付就行
2005-2015 外贸红利 开厂做出口的老板 有个工厂就行
2010-2018 互联网爆发 早期进BAT的程序员 会写代码就行
2015-2020 自媒体/短视频 早期内容创作者 敢拍就行
2017-2021 加密货币早期 早期入场者 敢买就行

每一代人都有属于自己的电梯。关键是：你能不能看到电梯在哪，以及你愿不愿意走进去。

第三部分：吃到红利的人犯的最大错误

把电梯当成自己的腿。

这不是我编的，你在现实中一定见过这些人：

  • 2005年买了几套房的大叔，教年轻人”赶紧贷款买房，房价永远涨”
  • 90年代下海赚了钱的老板，觉得年轻人”不够努力，吃不了苦”
  • 那一代轻松拿到教职的教授，在社交媒体上教年轻人”人生最优策略”

他们的共同点是：把时代给的运气，错认为自己的实力；把特定窗口期的经验，当作永恒不变的规律。

心理学里有个概念叫”自利归因偏差”——成功时归因于自己的能力，失败时归因于外部环境。吃到红利的人尤其容易犯这个错误，因为他们确实”做了选择”（买房、考博、下海），所以真心觉得是自己的选择创造了成功。

但他们没有意识到：同样的选择，换一个时间窗口执行，结果可能天差地别。

  • 2005年买房 = 财富自由
  • 2021年买房 = 负资产套牢
  • 2000年读博留校 = 轻松当教授
  • 2024年读博毕业 = 找不到教职送外卖

选择一样，时代不一样，结果完全相反。

第四部分：为什么长辈的建议在今天会害死人

这里有一个结构性的认知错位：

长辈的建议 = 红利期的最优策略
年轻人的现实 = 红利消失后的零和博弈

用博弈论的术语说：

  • 红利期是正和博弈——蛋糕在变大，所有人都能分到增量。买房赚、不买房的工资也在涨。策略容错率很高，犯了错环境会帮你兜底。

  • 红利消失后是零和甚至负和博弈——蛋糕在缩小，一方的获得等于另一方的损失。买错房就是亏，选错行业就是浪费生命。策略容错率极低，犯一个错可能十年翻不了身。

长辈在正和博弈里形成的经验，搬到零和博弈里就是毒药。

举几个具体的例子：

长辈的建议 红利期效果 现在的效果
赶紧买房 躺赚三倍 套牢三十年
考公务员铁饭碗 稳定+隐形福利多 降薪+内卷到极致
找个稳定工作别折腾 一份工作干到退休 35岁被优化，技能过时
嫁个条件好的人 男方收入年年涨 男方随时被裁，房贷断供
读个博士当教授 毕业即留校 毕业即失业

他们不是坏人，但他们的善意建议，基于的是一个已经不存在的世界。

第五部分：当下的红利在哪里

既然旧的电梯已经停了，那新的电梯在哪？

我没有标准答案，但可以分享我的观察框架：

判断红利的几个信号：

  1. 行业增速远快于人才供给 → 门槛会被迫降低
  2. 传统人才不愿意或不敢进入 → 竞争者少
  3. 监管尚未完全跟上 → 机会空间大
  4. 需要跨界能力组合 → 纯科班出身反而没优势

我进网络安全时，信号是：国家推网络安全法，企业被迫合规，人才缺口是现有从业者的好几倍。我进Web3时，信号是：传统安全人才对加密货币有偏见不愿来，但交易所对安全的需求是刚性的。

红利永远存在，只是换了形式和位置。问题不是”还有没有机会”，而是”你能不能看到它，以及你敢不敢走进去”。

第六部分：红利思维 vs 存量思维

真正区分人命运的，不是学历、不是背景、不是起点，而是两种思维模式的差异：

存量思维（大多数人）：

  • 在现有赛道上和别人拼努力
  • 相信”只要足够努力就能胜出”
  • 追求确定性，走”被验证过的路”
  • 结果：在红利消失的赛道上越卷越惨

红利思维（少数人）：

  • 寻找新赛道开启的窗口
  • 相信”选择比努力重要”
  • 拥抱不确定性，走”还没被挤满的路”
  • 结果：在新窗口期获得超额回报

我从油画跨到安全，不是因为我画画比别人差，也不是因为我技术比别人强。是因为我看到了一扇窗正在打开，而大多数人还在挤那扇已经快关上的门。

第七部分：为什么我选择”退出”

2023年离开360后，我做了一个很多人觉得疯狂的决定：放弃社保，学雅思，拿香港身份，进Web3。

从”存量思维”看，这是自毁前程——放弃社保、离开大厂、去一个”不靠谱”的行业。

但从”红利思维”看，逻辑很清楚：

  • 中国互联网行业的红利期已经结束（裁员潮就是证据）
  • 社保是一个正在贬值的承诺（老龄化+养老金缺口）
  • 香港身份 = 全球流动性的期权
  • Web3安全 = 新窗口期的新赛道

我的前同事们还在大陆大厂996，有些人背了30年房贷。他们不是不努力——他们比我努力得多。但他们站在一部已经停了的电梯上，再怎么踮脚也上不去了。

努力很重要，但方向比努力重要一万倍。在错误的电梯上再怎么努力，也只是原地踏步。

结尾：给还在旧电梯上的人

如果你正在被长辈催着买房、催着考公、催着结婚生子，如果有”教授”在社交媒体上告诉你”嫁个有钱人就是女性的最优解”——请先问自己一个问题：

给你建议的那个人，他们的成功有多少是自己的能力，有多少是时代的电梯？

如果一个人在2000年轻松拿到了博士和教职，她有什么资格告诉2025年的年轻人”照我的路走就行”？她坐的那部电梯，早就停运了。

时代红利的残酷之处在于：

  • 吃到的人以为是自己的本事
  • 没吃到的人以为是自己不够努力
  • 两边都没意识到，决定命运的从来不是个体的努力程度，而是你站在哪部电梯上

别听已经上了楼的人告诉你怎么爬楼梯。去找还在运行的电梯。

我能给的唯一建议是：保持流动性，保持对新事物的敏感度，不要把自己锁死在任何一个正在下沉的系统里。电梯永远在换位置，你要做的不是死守一部，而是随时准备换乘。

至于那些站在停运电梯上、居高临下教你做人的”过来人”——微笑点头，然后转身走向那部他们看不见的新电梯。
