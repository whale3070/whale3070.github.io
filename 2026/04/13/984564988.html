​Genesis: Code the New Era
​创世纪：代码新纪元

📅 活动时间线
​
​开幕式 2025年4月21日 香港
初赛开发2025年4月 - 6月
初赛评审2025年6月下旬
决赛 & 颁奖2025年7月

​🏆 双赛道设置
​🤖 AI Track
​人工智能驱动的创新应用，如：
• ​AI Agent 与智能体生态
• ​AI 智能工作流开发
• ​生成式 AI 工具与平台
​⛓️ Web3 Track
​区块链技术原生应用，如：
• ​DeFi 协议与基础设施
• ​AI agent交易系统
• ​跨链与互操作性方案
​💡 特别鼓励：AI × Web3 融合项目、Vibe Coding 实践者

​💰 奖金池
​膨胀制奖金池，初始$5,000 USD 起，持续膨胀中。关注动态以第一时间知悉。
​更多赞助与奖金细节即将公布

​👥 参赛资格
​基础要求
​✓ 团队参赛，每队 2-5人
✓ 个人开发者可报名，主办方协助组队
✓ 不限背景，欢迎开发者、设计师、产品经理、研究员跨界组队
​技术要求
​✓ 作品需结合 AI 或 Web3 技术栈
✓ 决赛 Demo 需现场可演示
✓ 代码开源（赛后需提交 GitHub 仓库）
​特别鼓励
​🎨 Vibe Coding — 使用 AI 编程工具（Cursor、Windsurf、Lovable 等）辅助开发
🤖 AI Agent 应用 — 智能体与 Web3 场景深度结合
🌏 亚太区开发者优先 — 决赛需赴香港现场

​

​
