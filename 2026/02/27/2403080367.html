数码港创业家计划 (2025)

这份文档涵盖了 碼港特別關注創業家在不同階段的需要，提供一系列針對性培育計劃，包括人才培訓、種子資金、協助產品投入市場及進行投資者配對。

数码港创意微型基金（香港计划）—— 申请与执行全案整理

https://www.cyberport.hk/zh-cn/entrepreneurship/hong_kong_programme/

申请链接： https://ems.cyberport.hk/form/form_list

一、基金基本信息

  • 资金额度：港币 100,000 元

  • 执行周期：6 个月

  • 支持方向：数码科技创新概念、早期初创，验证市场需求、加速发展

二、申请资格

  1. 公司名义

    • 香港注册有限公司（或正在注册中）

  2. 个人名义

    • 主要申请人持有香港居民身份证

三、评审标准（总计 100%）

  1. 项目管理团队素质及能力：30%

  2. 运营模式及上市/落地时间：30%

  3. 项目创意及创新元素：30%

  4. 社会责任：10%

四、资金发放节点

  1. 签署基金协议后：发放 HK$10,000

  2. 通过中期评估 + 提交中期报告：发放 HK$45,000

  3. 通过终期评估 + 提交终期报告 + 完成项目：发放 HK$45,000

数码港培育计划（含数码港智慧生活初创培育计划）详解

数码港培育计划为旗舰创业支援计划，其中包含数码港智慧生活初创培育计划，旨在为数码科技领域的早期初创企业提供全方位培育支援，助力企业将创新方案转化为市场产品、拓展业务版图，推动香港创新科技产业发展。受培育公司不仅能获得财务资助，还可享受多维度创业支持，加速实现商业价值。

一、计划核心福利

受培育公司在24个月培育期内，可享受以下核心支援，全方位助力业务发展：

  • 财务资助：最高港币50万元，用于支持企业日常运营、产品研发等核心业务环节；

  • 驻场租金补贴：高达港币20万元，减轻企业办公场地成本压力，驻场培育公司还可使用免租工作间及数码港各类设施；

  • 全方位创业支援：包括与投资者、企业伙伴、专业服务提供者的资源对接，精心设计的创业培训与指导，商业配对、海外展会参与机会，以及创业导师一对一辅导等，覆盖初创发展全阶段。

二、申请资格

申请企业需同时满足以下所有条件，确保具备相应的发展潜力与执行能力：

  1. 主体资格：必须为香港注册（或正在注册成立）的科技有限公司，若处于注册过程中，须在申请截止日期起30个日历日内完成注册；且在本培育计划申请截止日期时，公司成立时间少于7年；

  2. 商业计划：具备可行的商业计划，其产品/服务可在12至18个月内开始市场营销；

  3. 股权要求：公司的创立团队须合法且实际持有超过51%的公司股权，或拥有公司控制权；

  4. 本地运营：成功入选计划的受培育公司，在计划期间至少有一名本地授权代表（主要负责本地运营和产品开发）实际在香港工作，授权代表可为创始人/联合创始人，或公司雇佣的本地全职员工；

  5. 其他限制：主要申请人不可同时担任其他香港培育机构（如香港科技园、香港设计中心）相关培育计划的受资助公司创始人、联合创始人或董事；项目不得同时接受其他公营资助机构的同类资助；若同一项目同时申请本计划与数码港创意微型基金，仅考虑本计划申请。

三、评审标准及比重（总计100%）

由包含风险投资者、学者、行业前辈及行业协会成员组成的独立评审小组，按以下标准分阶段评审，确保筛选出有潜力的优质初创企业：

  • 市场可行性及里程计划、对数码港策略群组的贡献：30%；

  • 项目管理团队的素质及能力：20%；

  • 业务的扩展性：20%；

  • 试验原型或安全设计产品是否解决真实问题：20%；

  • 创新性及社会效益：10%。

四、资金发放详情（总额最高港币50万元）

资金分阶段发放，与培育进度、检讨结果挂钩，确保资助有效用于企业发展，具体节点如下：

  1. 初始运营资金：发放港币10万元，用于企业启动阶段的基础运营；

  2. 阶段性资金（共40万元）：剩余40万元资金按每六个月的进度检讨结果分阶段发放，具体如下：

    • 第一阶段（第6个月）：完成进度检讨后，发放港币5万元；

    • 第二阶段（第12个月）：完成进度检讨后，发放港币5万元；

    • 第三阶段（第18个月）：完成相关绩效驱动评估及进度检讨后，发放港币10万元；

    • 第四阶段（第24个月）：完成进度检讨及项目培育后，发放港币10万元。

五、培育周期与阶段划分

整体培育周期为24个月，分为四个阶段，每个阶段均需完成进度检讨，确保企业稳步发展：

  1. 第一阶段：0-6个月（基础培育期，完成首次进度检讨）；

  2. 第二阶段：7-12个月（成长培育期，完成第二次进度检讨）；

  3. 第三阶段：13-18个月（提升培育期，完成绩效驱动评估及第三次进度检讨）；

  4. 第四阶段：19-24个月（成熟培育期，完成第四次进度检讨及项目收尾）。

数码港加速支援计划

为尚未准备开拓海外市场及寻求投资的数码港培育公司或已毕业的培育公司，提供最高达30万港元的资助额度，申请者必须成功获得认可的加速器计划录取。

资助内容包括：

  • 75%的加速器计划费用津贴；

  • 75%的办公室租金津贴；

  • 实习生50%的薪金津贴；

  • 75%的旅费和住宿补贴；

  • 75%的市场及推广费用津贴，最高不超过50000港元；

  • 75%的专业服务津贴。

申请条件：

  • 必须为目前参与或毕业于数码港创意微型基金或数码港培育计划的受培育公司；

  • 并成功获得数码港认可的加速器计划录取。

香港科技券计划（TVP）核心信息

香港科技券计划（TVP）是创新科技署推出的数字化转型资助计划，新公司可申、无营业额门槛，最高资助60万港元，按**75%**比例覆盖数字化开支。

一、核心资助规则（2026年最新）

  • 资助比例：政府资助75%，企业自付25%（3:1配对）
  • 累计上限：每家企业60万港元，最多分6个项目申请
  • 单项目上限：10万港元，项目周期12个月内完成
  • 申请模式：先用后付，获批后可申请25%预付款

二、申请资格（新公司友好）

  • 香港注册/登记的非上市公司，非政府资助机构及其附属公司
  • 在港有实质业务运作（提供近3个月银行账单、租约、合同等证明）
  • 无营业额、盈利、办公室实体要求，新成立公司可直接申请

三、可资助的数字化开支（网站/APP/系统全覆盖）

  • 科技方案：网站/APP开发、ERP、CRM、POS、云端系统、大数据、AI、区块链、物联网、网络安全方案等
  • 服务类：科技顾问、定制/现成软件订阅、设备租赁、项目审计（超5万港元项目，审计费上限3000港元）
  • 不可资助：厂房购置、日常运营、品牌VI、商标注册、内销拓展、生产线自动化设备等

四、申请流程（全年开放）

  1. 准备：供应商报价、诚信确认书、业务证明、项目建议书
  2. 提交：通过科技券计划管理系统在线申请
  3. 审批：创新科技署初审→TVP委员会评审（约6-8周）
  4. 签约：获批后签资助协议，申请25%预付款
  5. 执行：12个月内完成项目
  6. 结案：完成后2个月内提交报告+支出凭证，审核后发剩余75%资助

五、关键注意事项

  • 同一时间仅可执行1个项目，完成结案后才能申请下一个
  • 项目需与企业业务高度相关，定制化内容不超项目成本50%
  • 不可与其他本地公共资助重复申请同一项目

香港中小企业市场推广基金（EMF）

核心信息（2026最新）
EMF由香港工业贸易署管理，最高资助100万港元，用于企业境外展会、商务考察、客户对接等出口推广活动，新公司可申、无营业额门槛。

一、核心资助规则（2026年，有效期至6月30日）

  • 资助比例：政府资助50%，企业自付50%（1:1配对）
  • 单次上限：10万港元（取“50%核准开支”与“10万港元”较低者）
  • 累计上限：每家企业100万港元；其中网站/APP优化类累计不超50万港元
  • 申请模式：先用后付（实报实销），可提前申请50%预付款

二、申请资格（新公司友好）

  • 香港按《商业登记条例》登记的非上市企业
  • 在港有实质业务运作（提供近3个月银行流水、租约、合同等证明）
  • 非推广活动主办方/协办方/服务供应商及其关联公司
  • 无营业额、盈利、成立年限要求，新成立公司可直接申请
  • 中小企业定义：制造业员工＜100人；非制造业员工＜50人

三、可资助的推广开支（境外展会/考察/客户对接全覆盖）

  • 展会类：境外实体展会、香港本地面向境外市场的展会、线上虚拟展会（展位费、搭建、运营、嘉宾费）
  • 商贸考察：境外/线上商贸考察团（成员往返交通、住宿、团费）
  • 客户对接：海外商务洽谈、客户拜访相关差旅与活动开支
  • 线上推广：国际B2B平台年费、Google/Facebook广告、SEO、多语言官网/APP优化
  • 广告投放：境外贸易刊物广告、目标市场精准营销

四、申请流程（全年开放）

  1. 准备：活动方案、供应商报价、业务证明、诚信确认书
  2. 提交：通过工贸署EMF系统在线申请
  3. 审批：约30个工作日
  4. 预付款：获批后可申请50%预付款
  5. 执行：完成推广活动并保留所有凭证（合同、付款记录、现场照片等）
  6. 结案：活动结束后6个月内提交报销申请，审核后发剩余50%资助

五、关键注意事项

  • 同一时间可执行多个项目，无数量限制
  • 推广活动需与企业出口业务高度相关
  • 不可与其他政府资助重复申请同一项目
  • 政策有效期至2026年6月30日

BUD 专项基金

1. 资助额度

  • 每个项目最高：700 万港元
  • 资助比例：项目成本的 50%
  • 用途：拓展内地市场 或 一带一路市场

2. 你这条信息里的基本门槛

  • 公司成立：≥1 年
  • 年营业额：≥100 万港元
  • 本地员工：2–3 名（香港本地雇员）

3. 官方真实硬性门槛（必满足）

  1. 香港注册公司
  2. 有实质业务运作在香港
  3. 非上市、非政府机构/补贴机构
  4. 项目内容必须是：
    • 市场推广
    • 产品升级/认证
    • 参加展会/电商平台
    • 建立内地/一带一路销售点/物流等

4. 你能立刻自查的 3 个问题

  1. 公司满 1 年了吗？
  2. 近一年营业额 ≥ 100 万 HKD？
  3. 有 2–3 名香港强积金员工？
    → 三条都满足 = 基本资格达标

5. 适合你这类创业者的重点

  • Web3 / RWA / 科技类项目可以申请，只要业务能和「拓展内地/一带一路市场」挂钩
  • 可申报费用举例：
    • 内地/海外平台入驻、推广投放
    • 产品认证、合规、设计包装
    • 参展、差旅、宣传物料、网站/APP开发

除了这些还有其他遗漏的，可以申请的吗

结合香港针对科技初创（尤其RWA防伪溯源这类「数码+实体经济」项目）的资助体系，除了你已梳理的CCMF、培育计划、TVP、EMF、BUD、科技园Ideation外，以下是易遗漏但高度适配的资助/支持计划，按「直接资金资助」「成本补贴/资源支持」「专项赛道扶持」分类整理，附申请优先级和适配场景：

一、直接资金资助类（纯拿钱，无配比/低配比）

1. 数码港深科技创业计划（DeepTech Venture Programme）

  • 核心额度：最高100万港币（直接拨款）+ 额外资源包
  • 适配场景：RWA防伪溯源若涉及「区块链+供应链深科技」「AI+溯源算法」等深科技方向，而非纯模式创新，优先申这个（比普通培育计划额度更高）。
  • 关键门槛：香港注册公司、成立＜7年、核心技术有壁垒、团队有深科技研发能力。
  • 优势：除资金外，对接深科技投资人、实验室资源，适合RWA项目技术攻坚阶段。

2. 创意香港「创意智优计划」（CreateSmart Initiative, CSI）

  • 核心额度：最高500万港币（直接拨款，资助比例最高80%）
  • 适配场景：RWA防伪溯源项目若涉及「创意设计+溯源可视化」「非遗/品牌溯源+创意包装」（比如奢侈品/农产品溯源的创意呈现），可申报。
  • 关键门槛：香港注册公司、项目需体现「创意产业+科技融合」、有明确的创意产出（如溯源视觉系统、交互设计）。
  • 易遗漏点：很多科技创业者忽略「创意+科技」的融合申报角度，RWA溯源的「用户端可视化设计」可纳入申报范畴。

3. 劳工及福利局「青年发展基金」（YDF）

  • 核心额度：最高200万港币（种子资金，直接拨款）
  • 适配场景：核心团队≤35岁的青年创业团队，RWA防伪溯源项目面向「青年创业+供应链数字化」方向。
  • 关键门槛：团队成员50%以上≤35岁、香港注册公司、项目有社会价值（如助力中小商户溯源数字化）。
  • 优势：对营业额/成立年限无硬性要求，纯青年创业扶持，可与CCMF/培育计划叠加（不同开支）。

二、成本补贴/资源支持类（先花后报，高比例覆盖）

1. 中小企业发展支援基金（SDSF）

  • 核心补贴：最高10万港币（100%报销），或最高50万（75%报销）
  • 适配场景：RWA防伪溯源项目的「专业服务开支」（如法律合规、知识产权注册、税务咨询）——很多创业者漏报这类隐性成本。
  • 关键门槛：香港注册中小企业（员工＜100人/营业额＜5000万）、在港实质运营、开支与业务直接相关。
  • 易遗漏点：RWA项目涉及的「区块链专利注册」「跨境溯源合规咨询」可100%报销（单项目10万内），比TVP覆盖更精准。

2. 职业训练局（VTC）「人才配对计划」

  • 核心支持：补贴实习生薪金80%（最高每月1.2万港币/人）+ 免费人才培训
  • 适配场景：RWA项目需要扩招「数码营销、区块链开发、供应链数据分析」类实习生，降低人力成本。
  • 关键门槛：香港注册公司、实习生为VTC在读/应届毕业生、签订至少6个月实习合同。
  • 优势：无资金上限（按人数补贴），适合RWA项目早期团队搭建，可与数码港加速支援计划的实习生补贴叠加。

3. 香港贸易发展局（HKTDC）「展会资助计划」

  • 核心补贴：境外展会开支50%报销（最高15万/次），香港展会30%报销
  • 适配场景：RWA防伪溯源项目参加「香港国际物流及供应链展」「一带一路数码科技展」等，补充EMF的覆盖盲区（EMF侧重出口，HKTDC侧重展会全场景）。
  • 关键门槛：香港注册公司、展会与业务相关、提前向贸发局备案。
  • 易遗漏点：EMF和HKTDC可叠加申报（比如EMF报50%，HKTDC补剩余部分，需拆分开支凭证）。

三、专项赛道扶持（RWA/供应链/区块链专属）

1. 香港金融管理局（HKMA）「数码港元创新计划」

  • 核心支持：最高500万港币（直接拨款）+ 监管沙盒测试资格
  • 适配场景：RWA防伪溯源项目若涉及「数码港元+供应链支付溯源」「央行数字货币（CBDC）+跨境溯源」，属于金管局重点扶持赛道。
  • 关键门槛：香港注册公司、有成熟的技术原型、与持牌金融机构合作（或有合作意向）。
  • 优势：专属Web3/RWA赛道，资金额度高，还能获得监管合规指导，适合有金融场景的RWA项目。

2. 工业贸易署「供应链数字化资助计划」

  • 核心补贴：数字化改造开支75%报销（最高200万港币）
  • 适配场景：RWA防伪溯源聚焦「传统供应链数字化升级」（如物流、仓储、跨境贸易溯源），比TVP更聚焦供应链赛道。
  • 关键门槛：香港注册中小企业、供应链相关业务占比≥50%、改造后能提升供应链效率（如溯源效率提升30%以上）。
  • 易遗漏点：TVP侧重通用数字化，这个计划是供应链专属，RWA项目申报成功率更高，额度也更高（200万vs TVP 60万）。
